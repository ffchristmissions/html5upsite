Instructions to clone, commit and push changes from VS Code to Github using the terminal

In File explorer, find the area you want to create the clone. 
Go to the repo that you want to clone in github and get the HTTPS URL (GitURL)
Then open up the terminal and type in git clone GitURL e.g. $ git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY

Make changes to your code as required and hit ctrl + s to save whenever

When you want to upload these changes, make sure your terminal is looking at the right file location

Type: 
git commit -am "Insert comments here"
git push origin main

Done
